PROJECT Realm


About

{PROJECT REALM} is a very light-weight version of Create Techified, with no custom mods but just pure kubejs scripts.

 

{PROJECT REALM} is mainly made for a server, but it can also be played in singleplayer.

 

If you were wanting to play a trimmed down version of Create Techified, this modpack might just be for you!
